HTML Cheatsheet

Tag NameDescription
<!DOCTYPE html>

Tells the web browser that the document contains HTML code
<html> </html>

the root of an HTML document, should be added after the !DOCTYPE declaration. All content/structure for an HTML document should be contained between the opening and closing tags.
<head> </head>

contains all the information the browser uses to interpret the html that is not displayed to the end user. (often containing the link to the CSS file)
<body> </body>

This section contains all the information that WILL be displayed to the user
<link>

This tag links all documents relevant to the HTML file. Most of the time this includes CSS files but an also include other files such as libraries or...
Example: <link rel="stylesheet" href="./style.css" type="text/css">
<a> </a>

adds a hyperlink to your document, URL should be included in opening tag and what you want to call it should be between the tags. If target="_blank"the link will open up in a new window/tab.
Example: <a href="www.YourLinkGoesHere">Click Here For Link</a>
<h1> <h2> <h3>...</h*>

These are headings, they should always be used in sequential order and no number should be skipped
<table> </table>

This is what you are looking at! The table tag declares that the data contained within is part of a table
<thead> <thead>

This tag will contain all the names or the collums

<tbody> </tbody>

This will contain all the data except the headers
<th> <tr> <td>

These are your tags used to create the actual table. th = Table Header, tr = Table Row, td = Table Data.
Note: Dont forget your closing tags!
<img>

Used to add an image, Requires no closing tag.
Example:<img src="path/to/image" alt="Description of image" />
<video> </video>

This is used to post a video to the webpage.
Example:<video src="Link to video" controls>Video not Supported</video>
<audio> </audio>

This is used to post an audio file to the webpage.
Example:<audio src="Link to Audio" controls>Audio not Supported</Audio>
<!-- COMMENT -->

This is how you add comments, The text within the dashes will not be displayed and will not affect the functionality of the code
<button> </button>

The button tags add a clickable button to your webpage, whatever you put between the tags will be displayed on your button

Thanks for Checking it Out!
